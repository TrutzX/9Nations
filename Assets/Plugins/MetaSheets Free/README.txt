-----------------------------------------------------
               Quickstart
-----------------------------------------------------
From the Unity File menu choose
[Assets] > [MetaSheets]
1.) Create a new Google spreadsheet or open an
    existing one
2.) Enter in the first row your property names.
	From the 2nd row and onwards enter the values.
3.) Publish the Google sheets document via 
    [File] > [Publish document...]
4.) Copy the spreadsheet document URL
5.) Paste the url into the Meta Sheets
    document ID field
6.) Enter a valid class name for example 'Data'
7.) Click "Generate 'Data'.cs" button
8.) Now you can access your data from any script
    via the generated static class.
